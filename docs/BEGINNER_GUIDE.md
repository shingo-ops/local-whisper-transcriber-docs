## ターミナルって何？

**ターミナル = コンピューターに文字で命令を出すアプリ**

Finderでフォルダを開くのと同じことを、文字で行います。

**例:**
- Finderでフォルダを開く → ターミナルでは `cd フォルダ名`
- Finderでファイルを見る → ターミナルでは `ls`

---

## ターミナルの開き方

### 方法1: Spotlight検索（おすすめ）

1. **キーボードの `Command (⌘) + スペース` を押す**
2. **「ターミナル」と入力**
3. **Enter を押す**

```
🔍 Spotlight検索窓
┌────────────────────────┐
│ ターミナル             │ ← ここに入力
└────────────────────────┘
  ターミナル.app        ← これが表示される
```

### 方法2: Launchpadから

1. **Launchpad を開く**（Dockのロケットアイコン）
2. **「その他」フォルダ を開く**
3. **「ターミナル」をクリック**

### 方法3: Finderから

1. **Finder を開く**
2. **「アプリケーション」→「ユーティリティ」**
3. **「ターミナル」をダブルクリック**

---

## ターミナルが開いたら

こんな画面が表示されます：

```
Last login: Thu Mar 12 16:49:23 on ttys010
tanizawashingo@tanizawngonoAir ~ %
```

**各部分の意味:**

```
tanizawashingo  ← あなたのユーザー名
@
tanizawngonoAir ← あなたのMacの名前
~               ← 今いる場所（~ = ホームフォルダ）
%               ← ここから入力できる
```

## 初回セットアップ（初回のみ）

### ステップ1: プロジェクトフォルダに移動

**コマンドをコピー:**

```bash
cd ~/sales-ops-with-claude/02_apps/whisper-tts-tool/local-whisper-ollama
```

**ターミナルに貼り付け:**

1. 上のコマンドを **マウスでドラッグして選択**
2. **Command (⌘) + C** でコピー
3. **ターミナルをクリック**
4. **Command (⌘) + V** で貼り付け
5. **Enter を押す**

**表示が変わります:**

```
tanizawashingo@tanizawngonoAir local-whisper-ollama %
                              ↑
                              フォルダ名が変わった！
```

これで正しい場所に移動できました✅

---

### ステップ2: セットアップ実行

**コマンドをコピー:**

```bash
./setup.sh
```

**ターミナルに貼り付けて Enter を押す**

**こんな画面が表示されます:**

```
==========================================
完全ローカル会議文字起こしツール - セットアップ
==========================================

検出されたOS: Mac

【ステップ 1/6】Ollamaのインストール確認
```

---

### ステップ3: インストールの質問に答える

**質問1: Ollamaをインストールしますか？**

```
今すぐインストールしますか? (y/n):
```

**`y` を入力して Enter を押す**

```
y ← これを入力
```

**しばらく待つ（1-2分）**

文字がたくさん流れますが、**何もせず待つ**だけです。

**これも待つだけ（10-15分）**

進行状況（%）が表示されます：
- 3% → 10% → 30% → 50% → 100%

## もっと簡単に！エイリアス設定

毎回2つのコマンドを入力するのが面倒な場合、1回だけ設定すれば `meeting` と入力するだけで起動できます。

### 設定方法（1回だけ）

**以下をコピー&貼り付けて Enter:**

```bash
echo 'alias meeting="cd ~/sales-ops-with-claude/02_apps/whisper-tts-tool/local-whisper-ollama && ./run.sh"' >> ~/.zshrc
```

**次にこれを実行:**

```bash
source ~/.zshrc
```

### 設定後の起動方法

**ターミナルを開いて、これだけ:**

```bash
meeting
```

**Enter を押すだけでメニューが表示されます！**

## よくある質問

### Q1: コマンドって何？

**A:** ターミナルに入力する文字のこと

例:
```
cd フォルダ名    ← これがコマンド
./run.sh        ← これもコマンド
```

コピー&貼り付けで実行できます。

---

### Q2: コマンドのコピー&貼り付け方法は？

**コピー:**
1. マウスでテキストをドラッグして選択
2. Command (⌘) + C

**貼り付け:**
1. ターミナルをクリック
2. Command (⌘) + V
3. Enter

---

### Q3: `cd` って何？

**A:** **C**hange **D**irectory（ディレクトリを変更）の略

フォルダを移動するコマンドです。

**例:**
```bash
cd ~/sales-ops-with-claude
```

これで `sales-ops-with-claude` フォルダに移動します。

---

### Q4: `~` って何？

**A:** ホームフォルダを表す記号

```bash
~  =  /Users/あなたのユーザー名
```

**例:**
```bash
cd ~/Downloads
```

これは以下と同じ意味：
```bash
cd /Users/tanizawashingo/Downloads
```

---

### Q5: ターミナルを閉じても大丈夫？

**A:** はい、大丈夫です。

セットアップした内容は消えません。

次回使うときは、またターミナルを開いて `meeting` を実行するだけです。

---

### Q6: エラーが出たら？

**エラー例1:**
```
zsh: no such file or directory: ./run.sh
```

**原因:** 間違った場所にいる

**解決策:**
```bash
cd ~/sales-ops-with-claude/02_apps/whisper-tts-tool/local-whisper-ollama
./run.sh
```

---

**エラー例2:**
```
source: no such file or directory: venv/bin/activate
```

**原因:** セットアップがまだ完了していない

**解決策:**
```bash
./setup.sh
```

---

**エラー例3:**
```
zsh: command not found: meeting
```

**原因:** エイリアスがまだ設定されていない

**解決策:**
```bash
echo 'alias meeting="cd ~/sales-ops-with-claude/02_apps/whisper-tts-tool/local-whisper-ollama && ./run.sh"' >> ~/.zshrc
source ~/.zshrc
```

---

### Q7: 処理中に止まったら？

**Ctrl + C** で中断できます。

**手順:**
1. キーボードの `Control` キーを押しながら
2. `C` キーを押す

---

### Q8: もう一度最初からやり直したい

**解決策:**
```bash
cd ~/sales-ops-with-claude/02_apps/whisper-tts-tool/local-whisper-ollama
./setup.sh
```

セットアップを再実行すれば OK です。
